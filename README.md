# Projeto de Funções prontas - Olos Studio

## Descrição do Projeto
Este projeto tem como objetivo criar um conjunto de funções prontas para serem utilizadas em projetos de desenvolvimento de software. As funções serão desenvolvidas utilizando a linguagem de programação JavaScript e serão organizadas em um repositório no GitHub para fácil acesso e colaboração.

## Regras do Projeto
1. As funções devem ser escritas em JavaScript e seguir as melhores práticas de codificação.
2. Cada função deve ser documentada com comentários explicativos sobre seu propósito, parâmetros e retorno.
3. As funções devem ser testadas para garantir que estão funcionando corretamente (seguindo a nomenclatura de arquivos de teste `.spec.js`)
